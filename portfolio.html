Mi portfolio web

En mi portfolio web Front-end vas a encontrar diferentes ejercicios y ejemplos realizados con diferentes herramientas utilizadas para el desarrollo web.

Proyectos realizados con Wordpress, ejercicios de JavaScript, auditoría SEO, etc... son algunos de los contenidos que podrás ver en este portfolio.

Además podrás visitar mi primer proyecto web completo y real, llamado todocasinuevo.com.


Proyectos con Wordpress

En esta sección puedes acceder a tres ejemplos de webs creadas con uno de los CMS más populares actualmente para la creación de páginas web.

Por un lado, está la página de "Artesanía Vikinga", una joyería online, con diferentes páginas de categorías de productos y sus correspondientes filtros de producto.

La página de "Japan.com", una web sobre el país del sol naciente, con un blog y diferentes secciones temáticas como fotografía japonesa, manga, arte contemporáneo o arquitectura.

Finalmente, la web de "Ahora me ves", una empresa ficticia de diseño web y márketing.
