Crear páginas web con código

Para crear una página web sin utilizar ningún tipo de sistema de gestión de contenidos o CMS implica que debemos conocer una serie de lenguajes y tecnologías informáticas.

Cualquier profesional Front-end debe tener conocimientos de HTML5, CSS3 y JavaScript.


Qué es el HTML5

HTML5 son las siglas de HiperText Markup Lenguage (versión 5), lo que significa que es un lenguaje de marcas o etiquetas, destinado a crear una estructura básica de los elementos que van a componer una página web.

Entre algunas de las muchas etiquetas que existen dentro de este lenguaje, tenemos aquellas destinadas a representar párrafos (p), encabezados (header), títulos (h1, h2, ...), menús de navegación (nav), pie de página (footer), ...

Podríamos comparar estas etiquetas con los ladrillos, tuberías, tejas, etc... necesarios para construir una casa, pero que de por sí no son suficientes para obtener el resultado final desado.

Todos esos elementos deberán ser "colocados" de una determinada manera utilizando diferentes técnicas y después tendrán que ser "decorados" para que tengan una apariencia atractiva.

De esta última parte se encarga el CSS.


Qué es el CSS

CSS3 son las siglas de Cascade Style Sheet (versión 3), o lo que es lo mismo, hojas de estilo en cascada.

Se trata del lenguaje o tecnología destinada a "organizar" y "dar estilo" a los elementos de una página web (creados con HTML5).

En la práctica, esto significa que el CSS3 se encarga de que las imágenes, los títulos, los párrafos, los menús, etc... (que ya hemos creado con HTML5) estén colocadas en una determinada posición.

Además, también es el encargado de que los elementos tengan uno u otro tamaño, tipos de fuente, colores de las letras, etc...

Continuando con la analogía anterior, serían aquelas técnicas encargadas de ir colocando esos ladrillos, tuberías, etc... según el plano deseado, así como posteriormente decorar y pintar la casa según el diseño establecido.


Qué es JavaScript

El JavaScript es un lenguaje de programación que, a pesar de su nombre, no tiene nada que ver con otro lenguaje de programación lamado Java.

Concretamente se trata de un lenguaje muy por los profesionales Front-end y, por tanto, en la parte "cliente" de la web. Es decir, en la parte que va a ser visible por los usuarios de esa web.

La utilización de este lenguaje de programación es la que va a conferir a la web del dinamismos tan necesario hoy en día.

Con JavaScript se puede conseguir gran cantidad de "efectos" en la web tales como creación de elementos que no existen previamente en la estructura inicial, cambios de apariencia de elementos, y un largo etcétera.

Por concluir la analogía utilizada anteriormente, diríamos que el JavaScript es el que se encarga de abrir y cerrar la puerta del garaje, encendido de luces, sistemas de alarma, etc...


Ventajas e inconvenientes de crear páginas web con código

La principal ventaja de crear una página web utilizando las tecnologías descritas anteriormente radica en la total libertad y versatilidad a la hora de crear todo aquello que se desee sin limitaciones (salvo las establecidas por el propio código).

La utilización de algún CMS como Wordpress puede suponer la existencia de algún tipo de limitación establecida por los creadores de los diferentes temas y plugins que se utilizan en estas plataformas.

Por otro lado, para poder crear una web mediante código, son necesarios unos conocimientos técnicos importantes de dichas tecnologías y lenguajes, mientras que la creación de una web con Wordpress u otro CMS puede ser abordada con unos pocos conocimientos en informática básica.

Además, utilizando un CMS tenemos también resuelta la parte de Back-end que también resulta imprescindible en cualquier proyecto web.

La elección de una u otra opción para crear una página web es una decisión personal y depende de los conocimientos y capacidades de cada usuario.
